# 重庆大学本科毕业论文（设计）LaTeX 模板

本仓库是重庆大学普通本科毕业论文（设计）LaTeX 工程模板。模板基于学校提供的 LaTeX 版本整理，并结合本科毕业论文（设计）撰写规范化要求进行工程化拆分和格式对齐。

## 依据与参考的文件

本模板主要参考以下学校资料：

- `重庆大学普通本科毕业论文（设计）Latex模版-202401.zip`
- `重庆大学普通本科毕业论文（设计）撰写规范化要求（2023年修订）.doc`
- `重庆大学普通本科毕业论文（设计）模版_202312.doc`
- `附件：“顺序编码制”的参考文献的引用和著录格式.doc`

如学校后续发布新版规范，应以学校最新正式文件为准，并同步更新本模板。

## 调整目标

1. 尽量贴合学校 Word 版格式要求，减少 LaTeX 版与正式规范之间的细节差异。
2. 将常见格式项集中配置，避免在各章节中重复手工设置字号、页眉、编号和目录样式。
3. 保持工程结构清晰，便于个人书写、长期维护和开源传播。

## 仓库结构

- `main.tex`：论文主入口，负责按顺序加载配置与正文模块。
- `config/`：集中管理宏包、页面布局、字体、目录、页眉页脚和公共命令。
- `content/`：按论文组成部分拆分封面、摘要、目录、正文各章、参考文献、附录、致谢和声明。
- `assets/`：存放校徽、插图以及其它图片文件目录。
- `bib/`：预留 BibTeX 文献数据目录。

## 使用方式

1. 使用 XeLaTeX 编译，推荐命令：

    ```powershell
    latexmk -xelatex main.tex
    ```

2. 修改论文内容时，优先编辑 `content/` 下对应文件。
3. 修改全局格式时，优先编辑 `config/packages.tex` 和 `config/styles.tex`。
4. 替换图片时，将图片放入 `assets/` 下合适目录，并在正文中使用相对路径引用。
5. 提交前删除或忽略 LaTeX 编译中间文件，`.gitignore` 已包含常见生成物。

## 最后的说明

当前正文使用通用模板示例文字和官方示例图表，用于验证封面、摘要、目录、正文、图表、公式、参考文献、附录、致谢和声明页效果。正式写作时，请替换封面信息、摘要、正文、参考文献、附录和声明页中的占位内容。
