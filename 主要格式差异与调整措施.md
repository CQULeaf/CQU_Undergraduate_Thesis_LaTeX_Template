# 主要格式差异与调整措施

这份文档具体解释本仓库和学校原始 LaTeX 模板之间的主要差异，也说明这些差异我是怎么处理的。它更像说明书，重点是帮助你快速理解：

- 模板改了什么
- 为什么要这样改
- 这些改动放在哪些文件里
- 你后续应该改哪里

如果你是第一次使用本仓库，建议先看 README，再看这里。

> 当然其实提供的模板已经完全可以直接用了，这里只是为了给感兴趣的读者更清晰地呈现工作细节。

## 这份模板主要做了什么

1. 保留封面、摘要、目录、正文、参考文献、附录、致谢、声明这些完整环节。
2. 把学校提供的模板不符合学校规范的地方全部一一对照着补齐了。
3. 把原来容易散落在各处的格式设置集中到了配置文件里。
4. 把单文件工程拆成了更适合书写与长期维护的目录结构。

## 主要差异与调整方式

| 关注点 | 原始模板常见问题 | 本仓库的处理方式 | 你应该去哪里改 |
| --- | --- | --- | --- |
| 文档结构 | 内容集中在一个大文件里，不利于维护 | 入口、配置、正文分离 | `main.tex`、`config/`、`content/` |
| 页面布局 | 边距、装订线、页眉高度分散设置 | 统一在导言区配置 | `config/packages.tex` |
| 页眉页脚 | 目录、正文、附录页眉容易不一致 | 用统一的页眉页脚宏处理 | `config/styles.tex` |
| 前置页 | 摘要、目录、致谢等页的切换容易混乱 | 提供统一的前置页页面宏 | `config/styles.tex` |
| 正文章节 | 每章的分页和页眉需要反复手写 | 提供统一的章节页页面宏 | `config/styles.tex` |
| 字体字号 | 正文中直接写字号命令会很碎 | 提供正文基准字体宏 | `config/styles.tex` |
| 章节标题 | 标题样式分散，难以统一 | 使用 `titlesec` 集中定义 | `config/styles.tex` |
| 目录样式 | 默认目录层级和点线不一定合适 | 使用 `titletoc` 统一排版 | `config/styles.tex` |
| 图表编号 | 图、表、公式编号规则不统一 | 按章节编号 | `main.tex` + `config/packages.tex` |
| 图表标题 | 图题、表题样式容易偏离中文论文习惯 | 用 `caption` 统一控制 | `config/packages.tex` |
| 参考文献 | 引用方式与列表样式容易混在一起 | 保留顺序编码制的示例写法 | `content/references.tex` |
| 声明页 | 下划线、勾选框和目录锚点容易出问题 | 单独处理声明页格式 | `content/declaration.tex` |

## 文件分工

1. `main.tex`

    这里是总入口，只负责按顺序加载配置和各部分内容，不能在这里堆正文。

2. `config/packages.tex`

    这里放全局宏包和基础排版设置，包括：

    - 文档类
    - 页面边距
    - 字体
    - 图表与公式编号
    - 参考文献和超链接
    - 算法环境

3. `config/styles.tex`

    这里放所有“会反复用到”的格式规则，包括：

    - 章节标题
    - 目录样式
    - 页眉页脚
    - 正文基准字体
    - 前置页和章节页宏

4. `content/`

    这里放实际内容。现在里面是模板示例，正式写作时你主要改这里。

5. `assets/`

    这里放图片、校徽和后续可能会用到的图源文件。

6. `bib/`

    这里预留给文献数据库。当前模板默认使用 `thebibliography`，以后也可以切换到 BibTeX。

## 使用时怎么改

1. 先改封面信息。
2. 再替换摘要、正文、参考文献、附录和致谢。
3. 如果你发现整套页面风格都要调整，先看 `config/`。
4. 如果只是某一章内容不同，直接改 `content/`。
5. 如果你要换图，先放进 `assets/`，再在正文里引用。

## 编译提醒

推荐使用 XeLaTeX：

```powershell
latexmk -xelatex main.tex
```

如果你只改了正文内容，通常不需要动 `config/`。如果学校更新了格式要求，优先改 `config/packages.tex` 和 `config/styles.tex`，再检查 `content/` 是否需要同步。

## 最后的说明

这份模板的目标是做一个好用、好改、结构规范与内容完整的本科毕设 LaTeX 模板。不过需要注意的是，论文正式提交前，还是要按学院或导师要求做最后检查。
